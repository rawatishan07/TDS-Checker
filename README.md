# 💰 TDS Checker

> **Automated TDS validation based on Debit Balance Percentage.**

This tool automatically calculates the **Debit Balance Percentage** for each transaction and verifies whether it falls within the predefined **TDS (Tax Deducted at Source) ranges**. It then classifies the transaction into the appropriate TDS category, eliminating the need for manual verification.

---

## ✨ Features

- 📊 Calculates **Debit Balance Percentage**
- ✅ Validates transactions against predefined TDS slabs
- 🎯 Classifies records into the correct TDS category
- ⚠️ Flags unmatched or invalid records
- 🚀 Reduces manual effort and improves accuracy

---

## 📌 Supported TDS Ranges

| TDS Rate |
|----------|
| 🟢 0.01% |
| 🔵 2% |
| 🟣 2.1% |
| 🟠 4% |
| 🔴 10% |

---

## ⚙️ How It Works

1. 📥 Reads transaction data.
2. 🧮 Calculates the **Debit Balance Percentage**.
3. 🔍 Compares it with predefined TDS ranges.
4. 🏷️ Assigns the matching TDS category.
5. 📤 Generates the classified output.
